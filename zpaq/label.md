# ZPAQ
> Journaling archiver for maximum compression and data integrity.

## ✨ Highlights

* **High-Ratio Compression:** Optimized for extreme file size reduction.
* **Journaling System:** Incremental backup capability with built-in history tracking.
* **Deduplication:** Efficiently handles redundant data across multiple versions.

## 🚀 Quick Start

Basic command to create a compressed archive:

    zpaq add backup.zpaq ./files/

## 📝 About

ZPAQ is a free and open source incremental, journaling command-line archiver for Windows, Linux and Mac OS/X.

Unlike traditional formats, ZPAQ is designed for long-term data storage, offering sophisticated deduplication and a robust architecture that ensures your backups remain accessible and compact over time.
